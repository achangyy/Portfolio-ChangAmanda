VEXU Robotics

Objective:

Design and built the robot. Code the driver control and autonomous routines.

Push Back Game Context

Game is based on controlling goals and mangaging blocks in a 12'x12' area. There are high, middle, and low goals where you can score and maintain control to score. There is a 15 second autonomous period then a 1 minute and 45 second driver-controlled period.

What I did

- Design the first intake iteration from a slope to an S-shape to increase carrying capacity to score quickly

- Helped assemble drive train

- Built upon previous code due to more complex autonomous routines, addition of sensors+pneumatics, and adding more driver control features

- Discussed with mechanics on what should be fixed and gave them the brain port assignments

Tools Used

Fusion360, Visual Studio Code (C++), drill, wrench, screwdriver
